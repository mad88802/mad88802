🛠️ Animated Tech Stack SVG

This page previews the custom tech-stack.svg asset we built for your GitHub profile.

🚀 How to use this in your GitHub Profile

  1. Upload both README.md and tech-stack.svg to your GitHub profile repository (usually named <username>/<username>).
  2. Ensure they are in the same folder (or update the image reference path in README.md if you place the SVG in an assets folder).
  3. Commit the changes, and GitHub will render the animated SVG automatically!

Note: Because the SVG uses prefers-color-scheme media queries, it will automatically switch styles to match the viewer's GitHub light or dark theme!
